# sonaRnetcdf4

## Introduction

sonaRnetcdf4 is a package for read and write netCDF4 Files in the SONAR-netCDF4 convension.

## Installation

Install the latest version from GitHub:
    ```r
    remotes::install_github("arnejohannesholmin/sonaRnetcdf4")
    ```
